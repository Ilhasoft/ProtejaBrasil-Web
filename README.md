# Proteja Brasil  
  
## Configuration

You need to define the following keys as an environment variable:

- SECRET_KEY
- DEBUG
- ALLOWED_HOSTS
- LANGUAGE_CODE
- TIME_ZONE
- DATABASES ("default")
- DEFAULT_FROM_EMAIL
- SERVER_EMAIL
- EMAIL_HOST
- EMAIL_HOST_USER
- EMAIL_HOST_PASSWORD
- EMAIL_ALIAS
- EMAIL_USE_SSL
- EMAIL_USE_TLS
